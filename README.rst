nim_sdl2_samples
================

sample codes.

Dependencies
------------

Arch Linux / Manjaro Linux

.. code-block:: bash

   sudo pacman -S --noconfirm sdl2 sdl2_gfx sdl2_image

Ubuntu

.. code-block:: bash

   sudo apt-get install -y libsdl2-dev libsdl2-image-dev libsdl2-mixer-dev libsdl2-net-dev libsdl2-ttf-dev

Run examples
------------

.. code-block:: bash

   nimble build -Y
   ./bin/sample_001_window.nim
